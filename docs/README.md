## Installation

To install UltiTools, follow the steps below:

1. Download the addon ZIP file from the [Blender Market](https://blendermarket.com/products/ultitools).
2. Launch Blender and navigate to **Edit > Preferences**.
3. In the Preferences window, select the **Add-ons** tab.
4. Click the **Install** button.
5. Locate and select the downloaded ZIP file.
6. Click the **Install Add-on** button.
7. Enable the addon by ticking the checkbox next to its name.
8. Configure any additional settings if required.
9. Click the **Save Preferences** button.

UltiTools is now installed and ready to use.

<a name="features"></a>
## Features

UltiTools offers the following key features:

1. **Smart Symmetrize**: Perform actions that go beyond the capabilities of the Mirror Modifier. Mirror Shape Keys and bisect the mesh without mirroring the mesh itself.
2. **Mesh Data Combine**: Easily combine Shape Keys or Vertex Groups.
3. **Quick Sculpt Mask**: Save and load Sculpt masks using Vertex Groups, allowing for the application of selection as masks.
4. **Text Animation**: Bring your Text Objects to life using keyframes and Python scripting.
5. **Sprite Sheet Gen**: Generate sprite sheets directly within Blender.
6. **Multi Camera Render**: Render multiple cameras using unique frame ranges and output paths for each camera.
7. **Bonus Blend file**: Includes a showcase of how Text Animation works.

For more detailed information about each feature, refer to the addon's documentation or tooltips within Blender.
